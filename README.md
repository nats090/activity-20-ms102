# Secure Full-Stack Starter

A boilerplate React + Express app with OWASP Top-10 mitigations, secure headers, rate limiting, JWT-based auth, input validation, and safe HTML rendering.

## Setup

1. Clone this repo
2. Terminal One:
   cd backend
   npm install
   npm start
3. Terminal Two:
   cd frontend
   npm install
   npm start

   BOOM!
